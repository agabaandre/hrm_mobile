## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

HRM Attend is an Android Java project designed to manage HRM attendance. This project leverages OpenCV for advanced image processing tasks. The goal is to provide a robust and efficient solution for attendance management.

Here's why you might find this project useful:
* Focus on creating a solution that solves a specific problem in HRM attendance.
* Utilize OpenCV for advanced image processing tasks.
* Implement best practices in Android development.

You can suggest changes by forking this repo and creating a pull request or opening an issue. Thanks to all the contributors who have helped expand this project!

### Installation

Below are the instructions for setting up and building the HRM Attend project with OpenCV.

1. Clone the repo
   ```sh
   git clone https://github.com/agabaandre/hrm_mobile.git
   ```
2. Navigate to the project directory
   ```sh
   cd hrm_mobile
   ```
3. Sync the project with Gradle files
   ```sh
   ./gradlew build
   ```
4. Add OpenCV to your project:
   - Download OpenCV Android SDK from [OpenCV](https://opencv.org/releases/)
   - Extract the downloaded SDK and copy the `sdk` folder to your project's root directory.
   - Update `settings.gradle` to include OpenCV:
     ```gradle
     include ':sdk'
     project(':sdk').projectDir = new File(opencvsdk + '/sdk')
     ```
5. Sync the project with Gradle files again
   ```sh
   ./gradlew build
   ```
